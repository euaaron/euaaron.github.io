# Stiebler.tech

[pt-BR]
**2018**
Me chamo Aaron Stiebler, sou estudante de Sistemas de Informação no IF Sudeste MG e, apesar disso, atuei como bolsista de Produção Audiovisual logo no inicio da faculdade, quando fiz este repositório para me auxiliar no desenvolvimento da minha página pessoal.

O objetivo da página é servir como um blog, um portifólio, e como uma espécie de "carta de apresentação", afinal, minha bolsa não vai durar pra sempre e preciso pagar meu aluguel... A página também contará com informações sobre projetos dos quais futuramente pretendo transformar em realidade. Além disso, colocarei todos os exercícios da matéria de "Desenvolvimento Web", para o professor Marcelo não falar que eu não fiz! kkk

**2020**
Durante os meses seguintes ao desenvolvimento dessa página, meu foco foi em aprender estruturas de dados, sql e backend com java. Conforme fui me familiarizando com o universo da programação, percebi que esta página foi feita mal e porcamente, e por isso, estou estudando uma forma melhor de recriar o site mantendo a premissa de *Single Page Application* e que ainda assim possa ser hospedado no github pages. Este tutorial ensinando a [hospedar uma aplicação em Angular totalmente funcional no GitHub](https://blog.bitsrc.io/deploy-your-angular-project-to-github-pages-7cbacb96f35b), que encontrei dia 1º JUL, parece ser bastante promissor.
